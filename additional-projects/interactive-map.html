Interactive Map

Easily access tutorials & checklists for The Legend of Zelda: Breath of the Wild

Overview

Create a website that allows the user to track in-game progress for The Legend of Zelda: Breath of the Wild, and view tutorials if necessary.

Objectives

  • Site design should be heavily centered around the unique visual style of Breath of the Wild
  • Code the site from scratch using HTML, CSS and JavaScript
  • Avoid external links at all costs
  • Include tutorials and map icons for each of the 120 shrines scattered across Hyrule
  • Allow the user to zoom in and out of the map separately from the other site content
